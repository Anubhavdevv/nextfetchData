## Getting Started
First, install 
```bash
npm install

```
Second, run the development server:

```bash
npm run dev
# or
yarn dev
```

## Adding Credentials and after login

After login click on Create Task and then ask your questions 

Rrember to create env file for credentials of your firebase store
